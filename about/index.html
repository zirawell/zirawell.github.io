About

我是泽拉威尔，毕业于华东理工大学复合材料与工程专业，后转职 IT ，目前从事 Java 开发，居住于中国上海。个人比较喜欢旅游、摄影、剪辑和一切美好的事物。

写这个博客的目的是为了记录自己的生活以及消化所学到的知识，同时进行对外分享，希望可以结交到更多的朋友。

本站使用 Github Pages 和 Hugo 进行搭建，具体操作步骤在我的博客中有，当然你也可以下载我的源码进行修改。
