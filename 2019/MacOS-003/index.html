MacOS用Touch ID授权

Written by Zirawell with ♥ on June 16, 2019 in 效率提升

系统环境

  • 机型：MacBook Pro 13″ 2017

  • 系统：macOS 10.14.5

⚠️ 注意**

• 由于系统的功能是随着升级而不断变化的，所以本文所述的方法不保证长期有效。

操作方法

打开“终端”，执行以下命令：

sudo sed -i ".bak" '2s/^/auth  sufficient  pam_tid.so\'$'\n/g' /etc/pam.d/sudo

然后按要求输入您的管理员密码，回车。

该命令的作用是把 /etc/pam.d/sudo 备份为 /etc/pam.d/sudo.，然后在 /etc/pam.d/sudo 的第二行前面加入 auth sufficient pam_tid.so 这个字符串。

如果需要恢复原文件，请执行：

sudo mv /etc/pam.d/sudo.bak /etc/pam.d/sudo

最终效果

打开终端，执行 sudo su，然后就可以刷指纹啦！
